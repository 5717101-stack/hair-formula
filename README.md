# Hair Formula — Professional Color Calculator

Precision L'Oréal hair color formula calculator for professional hair stylists.

## Features

- **Formula Calculator**: Calculate roots and ends formulas with L'Oréal color rules
- **Client Management**: Save formulas to client profiles with full visit history
- **Mobile-First**: Designed for stylists holding their phones while mixing colors
- **PWA Ready**: Installable as a mobile app

## Tech Stack

- Next.js 16 (App Router, Static Export)
- Tailwind CSS + shadcn/ui
- Framer Motion
- Lucide React
- localStorage for client persistence

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Render as a static site. Auto-deploys from `main` branch.
